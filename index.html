DESARROLLO WEB:

HTML

ES UN LENGUAJE DE MARCADO EN EL QUE CONSISTE ARMAR A TRAVES DE UNA BASE DE ETIQUETAS Y MARCAS LA ESTRUCTURA DE UNA PAGINA WEB

CSS

ES UNA HOJA DE ESTILOS ENGARGADA DE DARLE FORMA AL ASPECTO GRÁFICO DE UN DISEÑO PARA QUE LA PAGINA WEB SEA VISIBLE ANTE EL USUARIO

JavaScript

ES UN LENGUAJE DE PROGRAMACIÓN QUE SE ENCARGA DE REALIZAR LA ESTRUCTURA DE METODOS Y FUNCIONES DENTRO DE UNA PAGINA WEB DANDOLE FORMA A LAS ACCIONES Y EVENTOS QUE SE GENERARAN DENTRO DEL ESQUEMA DINÁMICO.

REACT

ES UNA LIBRERIA DE JAVA SCRIPT QUE CONSISTE CREAR INTERFACES GRÁFICAS DE USUARIO FACILITANDO SU DESARROLLO EN UNA PÁGINA.

NODEJS

ES UN ENTORNO OPEN SOURCE MULTIPLATAFORMA PARA LA EJECUCIÓN DE LOS JAVASCRIPT ORIENTADO A EVENTOS ASÍNCRONOS. ES ÚTIL PARA LA CREACIÓN DE PROGRAMAS ESCALABLES.

REST-API

Una API REST es un backend capaz de contestar a las llamadas a una serie de URLs en formato JSON. Es utilizado por muchas empresas gracias a su versatilidad y eficiencia.

BOOTSTRAP

ES UNA LIBRERÍA FRONT-END DE CSS MULTIPLATAFORMA OPENSOURCE PARA DISEÑO DE SITIOS Y APLICACIONES WEB, YA SEA PARA MODELOS RESPOSIVES, GRID Y LAYOUTS. SEGÚN DIFERENTES OPINIONES QUE SE ENCUENTRAN EN LA WEB, ES LA FRAMEWORK MAS UTILIZADA.

MATERIALIZE

COMO BOOTSTRAP, ES UNA FRAMEWORK FRONT-END DE CSS MULTIPLATAFORMA OPENSOURCE BASADO EN MATERIAL DESING DE GOOGLE, QUE PERMITE EL DISEÑO DE SITIOS WEB.
